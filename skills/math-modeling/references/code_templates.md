# Python 代码模板规范

## 代码生成原则

1. **可直接执行**：生成的代码必须能立即运行，不缺依赖
2. **中间结果持久化**：所有数据处理步骤保存结果到本地文件（CSV/JSON/pickle）
3. **详细输出**：print/logging 输出关键计算步骤和结果
4. **复用前置任务输出**：优先读取前置任务生成的文件，不重复计算
